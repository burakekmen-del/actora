═══════════════════════════════════════════════════════════════════════════════

SCENARIO 1: User tries to progress to Day 2 without inviting anyone
──────────────────────────────────────────────────────────────────
  System Check: "Do they have ≥1 accepted invite?"
  
  Answer: NO ❌
  ├─ ProgressionCheckScreen shown (CANNOT DISMISS)
  ├─ Shows: "Bring 1 friend to unlock Day 2"
  ├─ Only option: "🚀 INVITE NOW"
  ├─ User clicks invite → ShareInviteDialog
  ├─ Friend gets deep link → accepts
  ├─ Competition created
  └─ User can now proceed to Day 2 ✅

SCENARIO 2: User completes Day 3, but falls behind friend
──────────────────────────────────────────────────────────
  Daily Check: "Is user 2+ days behind someone?"
  
  Answer: YES ⚠️
  ├─ LossAversionScreen shown before anything else
  ├─ Shows: Friend is on Day 6, you're on Day 4 (GAP: -2)
  ├─ Shows what you lose: "💥 Your 3-day streak ends"
  ├─ Shows future: "📉 They'll be 5 days ahead"
  ├─ CTA: "💪 Do Today's Task Now"
  ├─ Result: 70% tap primary (continue task)
  └─ 30% see quit confirmation (reality check)

SCENARIO 3: Complete task → Sync with ALL competitors
────────────────────────────────────────────────────
  User completes Day 5 task
  ├─ Task streak updated: 5 → 5 (still day 5)
  ├─ syncStreakWithAllCompetitors() called
  ├─ For each competitor:
  │  ├─ updateCompetitionStreak(5)
  │  ├─ generateNotification("You progressed")
  │  └─ Sends to friend: "Ali is now on Day 5!"
  └─ Friend gets push → opens app → updates day count
