# Actora App Store Metadata

## App Name

## Subtitle

## Short Description
Actora helps you build a daily execution habit with one clear task, simple streaks, and lightweight accountability.

## Long Description
Actora is a focused execution app built to help you complete one meaningful task per day.

Each day you get a single clear action, a streak that tracks consistency, and lightweight social accountability that makes it easier to keep going.

Actora is designed around simplicity:
- one task at a time
- a daily completion loop
- streak visibility for motivation
- optional challenge links to invite a friend

The goal is not to overwhelm you. It is to make follow-through easier and more consistent.

## Privacy Policy Summary
Actora stores only the minimum data needed for app functionality, such as streak state, daily completion state, and optional challenge invite metadata.

If analytics is enabled, it is used for product measurement and feature improvement.

No sensitive personal data is required for core use.
